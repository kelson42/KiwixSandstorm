All of Wikipedia (and much more!) on your home or community server, in a few clicks

With Kiwix, you can set up your own server to host a copy of many popular open data websites, in multiple languages, including:

  • Wikipedia (and other Wikimedia projects)
  • Project Gutenberg
  • Stack Overflow (and other Stack Exchange sites)
  • Ted Talks

See here for all available content. To learn more about the Kiwix project (including how to install Kiwix directly onto your phone or desktop instead of Sandstorm) see here.

What is this not?

This is not wiki software, where anybody can edit the content. Once you download and install the content, it is read-only.

Why do I want this?

You may find this useful if:

  • You want to search and view content privately.
  • You're just a data hoarder.
  • You have Sandstorm set up in a low connectivity environment, and you want quick access to popular content.
  • You are already a fan of the standalone Kiwix server, but you want the ease of use that comes with Sandstorm.

Unsupported features

This Sandstorm app is based on the Kiwix project, but is still in progress, so it may lag behind in some features. See here to read about current known limitations.
